
The Open2x Launcher
-------------------

The Launcher is the official Open2x user interface.


Design
------

Skin Support - Zip Files or directories.

Customizable - Ini Files

Complete - Rom Launcher, Settings, etc.

Scan the sd card for gpe, gpu. save results in ini file ?


Goals
-----

Ultimatly break off chunks of support code:

lib2x - General functionality, eg:
        * battery checker
        * serial number
        * encryption?
        * tv-out
        * ini file support

lib2xui - User interface library functions eg:
        * file selector
