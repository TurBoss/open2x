
 =====================================
 sdl 'hello world' sample for the gp2x
 =====================================

 1) Build the sample.

  make

 2) Install the sample to your sd card.

  make PREFIX='/media/GP2X/Demos/hello' install 

 3) Run the sample on the gp2x.

  Select 'game' from the menu.
  Select your sd card.
  Browse to the installation directory.
  Run the 'hello.gpe' program.
