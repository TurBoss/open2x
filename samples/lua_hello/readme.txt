
 =====================================
 lua 'hello world' sample for the gp2x
 =====================================

 1) Build the lua interpreter.

  make

 2) Install the sample to your sd card.

  make PREFIX='/media/GP2X/Demos/lua_hello' install 

 3) Run the sample on the gp2x.

  Select 'game' from the menu.
  Select your sd card.
  Browse to the installation directory.
  Run the 'wrapper.gpe' shell script.

 4) View the results.

  Select 'e-book' from the menu.
  Select your sd card.
  Browse to the installation directory.
  View the 'output.txt' file.
